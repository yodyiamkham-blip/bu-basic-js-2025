Objects in JS

  • An object is a collection of key-value pairs (properties)
  • Properties can store data (strings, numbers, arrays, other objects) or functions (methods)
  • Objects help organize related data together
const person = {
    name: "Alice",
    age: 25,
    greet: function() {
    console.log("Hello, my name is " + this.name);
    }
};

console.log(person.name); // Alice
person.greet();           // Hello, my name is Alice
