Document Object Model (DOM)

  • W3C Standard
  • the standard for accessing documents
  • W3C DOM standard can be separated into 3 different part
    • Core DOM - standard model for all document types
    • XML DOM - standard model for XML documents
    • HTML DOM - standard model for HTML documents
"The W3C Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document."

What is the HTML DOM?

The standard object model and programming interface for HTML. It defines

  • The structure of the document (as a tree).
  • The methods to access and manipulate elements.
  • The properties of each element (like .innerHTML, .value, .style, etc.).
  • The document is the root
  • HTML tags are branches
  • Attributes are the leaves

In other words: it is the standard for how to get, change, add, or delete HTML elements.

With HTML DOM, you can use JS to

  • Change content (document.getElementById("id").innerHTML = 'Hello';)
  • Change style (elenment.style.color = 'red';)
  • add or remove elements
  • React to events (like clicks, keypresses, etc.)

HTML DOM Document Object

To access any element in HTML page, always start with accessing the document object.

To Find HTML elements

  • document.getElementById(id): find element by element id
  • document.getElementsByTagName(name): find elements by element tag name
  • document.getElementsByClassName(name): find elements by element class name

To change HTML Element

  • element.innerHTML: change the innerHTML of an element
  • element.attribute: change the attribute value of html element
  • element.setAttribute(attribute, value): change the attribute value of html element

To add and delete elements

  • document.creteElement(element)
  • document.removeElement(element)
  • document.appendChild(element)
  • document.replaceChild(new, old)

To add events handlers

  • document.getElementById(id).onclick = function(){code}
  • document.getElementById(id).onchange = function(){code}
